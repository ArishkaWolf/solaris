# Solaris Next App

Интерактивный сайт, посвященный театральной постановке «Солярис» по мотивам одноименного произведения Станислава Лема.

Проект представляет спектакль через визуальные, аудио- и видеоразделы: баннер, описание постановки, блок с актерами и ролями, перевод монолога, музыкальный плеер, новостную ленту и дополнительные карточки с материалами.

## Функциональность

- Главный экран с названием спектакля и именем постановщика.
- Видео-блок с атмосферным фоновым роликом.
- Описание идеи и темы постановки.
- Раздел «Актеры и роли» с фотографиями, фактами и интерактивной кнопкой лайка.
- Блок с переводом монолога.
- Музыкальный плеер с треками, обложками, перемоткой, переключением песен и кнопками лайка/дизлайка.
- Новостная лента с видео.
- Дополнительные визуальные секции и карточки.
- Адаптивная структура на базе компонентов Next.js.

## Технологии

- [Next.js](https://nextjs.org/)
- React
- JavaScript
- CSS Modules
- Swiper
- HTML5 Audio / Video

## Установка и запуск

Установите зависимости:

```bash
npm install
```

Запустите проект в режиме разработки:
```
npm run dev
```
Откройте сайт в браузере:
```
http://localhost:3000
```
Доступные команды
```
npm run dev
```
Запускает локальный сервер разработки.
```
npm run build
```
Собирает production-версию проекта.
```
npm run start
```
Запускает собранную production-версию.

Структура проекта
```
solaris-next-app/
├── app/
│   ├── components/        # Компоненты интерфейса
│   ├── fonts/             # Локальные шрифты
│   ├── globals.css        # Глобальные стили
│   ├── layout.js          # Общий layout приложения
│   └── page.js            # Главная страница
├── public/
│   ├── actors/            # Фотографии актеров
│   ├── cards/             # Изображения для карточек
│   ├── songs/             # Музыка и обложки треков
│   └── ...                # Видео, иконки и другие медиафайлы
├── products.json          # Данные об актерах и ролях
├── tracks.json            # Данные музыкального плеера
├── package.json
└── README.md
```

## Основные компоненты
* Banner — главный экран с названием постановки.
* VideoBlock — видео-блок.
* Description — описание спектакля.
* Actors — карточки актеров и ролей.
* Translation — текстовый блок с переводом монолога.
* TracksSection — музыкальный плеер.
* NewsFeed — видеораздел новостной ленты.
* CardSection, FlipCard, FooterSection — дополнительные визуальные и информационные секции.

## Данные
Информация об актерах хранится в файле: products.json
Информация о музыкальных треках хранится в файле: tracks.json
Медиафайлы находятся в папке: public/

## Авторская идея
Сайт создан как цифровое сопровождение театральной постановки «Солярис». Он помогает зрителю глубже погрузиться в атмосферу спектакля, познакомиться с актерами, услышать музыкальные темы и увидеть дополнительные материалы постановки.
